# Kanban Board

Упрощённый аналог Trello. Позволяет создавать доски, колонки и карточки задач, перетаскивать карточки между колонками, добавлять описание, срок выполнения и изображения.

## Стек

Бэкенд: Python 3.11+, FastAPI, SQLAlchemy 2.0, SQLite, Uvicorn

Фронтенд: React 18, Redux Toolkit, @dnd-kit, Vite

## Возможности

- Регистрация и вход по имени пользователя
- Несколько досок на одного пользователя
- Создание, переименование и удаление колонок
- Создание карточек с названием, описанием и сроком выполнения
- Drag-and-drop карточек между колонками и внутри колонки
- Загрузка изображений к карточке, скачивание, удаление
- Подсветка просроченных карточек
- Обработка ошибок и состояния загрузки

## Запуск

### Требования

- Python 3.11+
- Node.js 18+

### Бэкенд

```bash
cd backend
# Windows:
python -m venv venv
venv\Scripts\Activate
pip install -r requirements.txt
# или вручную:
pip install fastapi uvicorn sqlalchemy pydantic python-multipart
uvicorn main:app --reload
```

Сервер запустится на `http://localhost:8000`.

База данных `kanban.db` и папка `uploads/` создадутся автоматически при первом запуске.

Документация API доступна по адресу `http://localhost:8000/docs`.

### Фронтенд

```bash
cd frontend
npm install
npm run dev
```

Приложение откроется на `http://localhost:5173`.

## Использование

1. Открыть `http://localhost:5173`
2. Ввести любое имя пользователя - если такого нет, аккаунт создастся автоматически
3. На главной странице создать доску
4. Внутри доски добавить колонки и карточки
5. Карточки можно перетаскивать между колонками и менять порядок внутри колонки
6. По клику на кнопку редактирования - открыть редактор карточки с описанием, датой и изображениями
